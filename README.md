# 🚀 Pomodoro Web App

---

## 📌 Overview

Pomodoro Web App adalah aplikasi berbasis web untuk meningkatkan produktivitas menggunakan teknik **Pomodoro**. Dirancang dengan pendekatan minimalis dan performa tinggi, aplikasi ini fokus pada satu hal: **membantu pengguna tetap fokus dan konsisten**.

---

## ✨ Features

### ⏱️ Pomodoro Timer Engine

* Default: 25 menit fokus / 5 menit istirahat
* Long break otomatis setiap 4 sesi
* Start / Pause / Resume / Reset
* Auto-next session (optional)
* Background timer (tetap jalan walau tab tidak aktif)
* Sound & browser notification

---

### 📝 Task Management

* Create / Edit / Delete task
* Estimasi jumlah Pomodoro per task
* Status: *Pending / In Progress / Completed*
* Attach sesi Pomodoro ke task

---

### 📊 Productivity Tracking

* Total waktu fokus harian
* Jumlah sesi selesai
* Riwayat aktivitas harian
* Streak produktivitas

---

### 📈 Analytics Dashboard

* Statistik harian & mingguan
* Visualisasi fokus (bar chart)
* Progress terhadap target harian

---

### 👤 User System

* Authentication (Email / Google)
* Custom setting:

  * Durasi fokus & break
  * Auto-start toggle
* Data tersimpan di cloud

---

### ⚙️ Smart Rules (Non-AI)

* Auto long break setiap 4 sesi
* Reminder saat idle
* Progress tracking berbasis target harian
